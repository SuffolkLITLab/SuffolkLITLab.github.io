1. Translate Technobabble

You need a tool to read and write code, and no, Microsoft Word won't do the trick. You need a text editor. Unlike word processors, they don't touch anything but text. Have you ever tryied to cut and paste something from Word only to have the formating all messed up? That's because there's a bunch of unseen stuff mixed in with your text. This works well for letters but no so much for code. Also, most text editors will highlight your code, making it easier to read. Some popular free text editors include Atom, Notepadd++ (Windows only), and Visual Studio Code. If you don't have a text editor, download one from the preceeding list. It will come in handy soon.

2. Read and Write Code

You need a tool to read and write code, and no, Microsoft Word won't do the trick. You need a text editor. Have you ever tried to cut and paste something from Word only to have the formatting all messed up? That's because there's a bunch of unseen stuff mixed in with your text. 3. Share Your Work on Github

Git is a system for managing the many versions of code one writes over the course of a coding project. At one level, it can act like track changes for computer code, allowing multiple people to collaborate on a single project. GitHub is a website on which you can host and manage projects using Git. It's also a community of users where people share their code.     6. Build Simple Expert Systems ("Chatbots")

    Most of the time when you interact with a "chatbot," you're interacting with a glorified decision tree. QnA is a markup language for people with little or no programming experience. Designed with attorneys in mind, it transforms blocks of text into interactive question and answer sessions (QnA interviews). Such rules-based systems are sometimes called expert systems, and the idea is that one can encode some small area of knowledge as part of an interactive decision tree, making it more broadly available. In the following lessons you will learn how to turn flowcharts into functioning QnA interviews/"chatbots."
